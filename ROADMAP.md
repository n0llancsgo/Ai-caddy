# AI Caddie roadmap

## Phase 1 - works on phone
- Player profile and club lengths
- Manual distance input
- Lie selection
- Deterministic club recommendation
- Voice readout
- Manual shot logging
- Basic round analysis

## Phase 2 - real course data
- Add real courses from OSM/API/manual editor
- Tee/green coordinates per hole
- Automatic nearby course selection
- Basic map view
- Distance to green center/front/back

## Phase 3 - sync and accounts
- Supabase Auth
- Sync profiles, rounds and shots
- Export round data as CSV
- Dashboard over many rounds

## Phase 4 - better caddie logic
- Safer tee target recommendations
- Personal miss tendencies per club
- Wind direction vs hole direction
- Competition mode vs training mode

## Phase 5 - wearables
- Meta Device Access Toolkit adapter
- Mock Device Kit tests first
- Voice-first experience
- Optional camera snapshots later

## Phase 6 - advanced
- Automatic shot detection
- Club tags / NFC
- Green maps / putting stats
- True green reading only after the core app is useful
